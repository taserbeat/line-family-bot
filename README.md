# LINE Family Bot

## 紹介

個人学習を目的に LINE bot を開発・運用していく。

## 使用技術・サービス

- LINE Developers、および関連する LINE プラットフォームサービス
- [GitHub](https://github.co.jp/)
- [Heroku](https://dashboard.heroku.com/)
- golang 1.16
